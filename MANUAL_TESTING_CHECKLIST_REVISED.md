## 附錄：測試數據準備

### 測試文檔準備清單
- [ ] 簡單合同.docx（10-20 條款）
- [ ] 示例協議.pdf（包含常見風險條款）
- [ ] 文本合同.txt（純文本格式）
- [ ] 大文檔.docx（10MB+，用於性能測試）

### 測試文檔應包含的內容
建議測試文檔包含以下類型的條款，以便測試風險識別功能：
- 違約責任條款（高風險）
- 保密條款（中風險）
- 知識產權條款
- 爭議解決條款
- 付款條款
- 終止條款
- 自動續約條款

### 示例風險條款
可以在測試文檔中加入以下內容：
```
1. "乙方承擔所有責任" - 高風險
2. "甲方有權單方面解除合同" - 高風險
3. "爭議提交北京仲裁委員會仲裁" - 中風險
4. "保密期限為永久" - 高風險
5. "合同自動續約，除非一方提前 90 天書面通知" - 中風險
```

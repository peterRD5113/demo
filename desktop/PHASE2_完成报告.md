## 🎯 关键技术亮点

### 1. 事务处理
版本保存使用数据库事务，确保：
- 版本记录创建
- 条款快照复制
- 版本标记更新
三个操作要么全部成功，要么全部回滚。

### 2. 防循环机制
同步滚动使用 `isSyncing` 标志：
```typescript
if (isSyncing) return; // 防止循环触发
setIsSyncing(true);
// 执行滚动同步
setTimeout(() => setIsSyncing(false), 50);
```

### 3. 性能优化
使用 `requestAnimationFrame` 优化滚动：
```typescript
requestAnimationFrame(() => {
  rightRef.current.scrollTop = leftRef.current.scrollTop;
});
```

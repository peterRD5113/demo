## ✅ 核心测试清单

### 测试1：保存版本 ⭐⭐⭐

2. 观察提示消息

**预期结果**：

- [ ] 无错误提示

### 测试2：进入对照模式 ⭐⭐⭐
**步骤**：
1. 点击"对照修订"按钮
2. 观察页面变化

**预期结果**：
- [ ] 按钮变为蓝色，文字变为"退出对照"
- [ ] 显示左右双栏布局

### 测试3：同步滚动 ⭐⭐⭐
**步骤**：
1. 在对照模式下滚动左栏
2. 观察右栏是否同步
3. 滚动右栏
4. 观察左栏是否同步

**预期结果**：
- [ ] 滚动左栏时，右栏同步滚动
- [ ] 滚动右栏时，左栏同步滚动
- [ ] 滚动流畅，无卡顿

---

### 测试4：退出对照模式 ⭐⭐
**步骤**：
1. 在对照模式下点击"退出对照"按钮

**预期结果**：
- [ ] 恢复单栏布局

---

### 测试5：修改后保存新版本 ⭐⭐⭐

3. **记录问题**：

   ```
   实际：...

   ```

### 常见问题

**问题1：点击"对照修订"没有反应**
- 检查：是否已保存过版本？
- 解决：先点击"保存版本"

**问题2：左右栏内容不一致**
- 检查：是否在保存版本后又修改了条款？
- 说明：这是正常的，左栏显示保存的版本，右栏显示当前编辑

**问题3：滚动不同步**
- 检查：Console 是否有错误？
- 尝试：刷新页面重试

## 🔧 技术实现细节

### API 调用
```typescript
// 获取最新版本
window.electronAPI.version.getLatest(documentId, userId)

// 获取版本条款
window.electronAPI.version.getClauses(versionId, userId)
```

### 滚动同步机制
```typescript
const handleLeftScroll = useCallback(() => {
  if (isSyncing || !leftRef.current || !rightRef.current) return;
  
  setIsSyncing(true);
  requestAnimationFrame(() => {
    if (leftRef.current && rightRef.current) {
      rightRef.current.scrollTop = leftRef.current.scrollTop;
    }
    setTimeout(() => setIsSyncing(false), 50);
  });
}, [isSyncing]);
```

### 条件渲染
```typescript
{!isComparisonMode ? (
  // 单栏模式
  <div className="clauses-section">...</div>
) : (
  // 双栏模式
  <div style={{ display: 'flex', gap: '16px' }}>
    <div ref={leftRef}>...</div>  {/* 左栏 */}
    <div ref={rightRef}>...</div>  {/* 右栏 */}
  </div>
)}
```

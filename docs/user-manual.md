## 系統介紹

### 什麼是合同風險管理系統？

合同風險管理系統是一款智能化的桌面應用程序，專為企業和法務人員設計，用於快速識別和管理合同中的潛在風險。

### 主要功能

- ✅ **智能風險識別**: 自動掃描合同條款，識別高、中、低風險
- ✅ **多格式支持**: 支持 DOCX、PDF、TXT 等多種文檔格式
- ✅ **項目管理**: 按項目組織管理多個合同文檔
- ✅ **批註功能**: 為風險條款添加批註和建議
- ✅ **報告導出**: 生成專業的風險分析報告（PDF 格式）
- ✅ **版本管理**: 追蹤文檔修改歷史
- ✅ **離線使用**: 完全本地化，無需網絡連接

### 系統要求

**最低配置**:
- 操作系統: Windows 10 / macOS 10.13 / Ubuntu 18.04 或更高版本
- 處理器: Intel Core i3 或同等處理器
- 內存: 4GB RAM
- 硬盤空間: 500MB 可用空間

**推薦配置**:
- 操作系統: Windows 11 / macOS 12 / Ubuntu 22.04
- 處理器: Intel Core i5 或更高
- 內存: 8GB RAM 或更高
- 硬盤空間: 1GB 可用空間

## 安裝指南

### Windows 安裝

1. 下載 `ContractRiskManagement-1.0.0 Setup.exe`
2. 雙擊安裝文件
3. 按照安裝向導完成安裝
4. 安裝完成後，桌面會自動創建快捷方式

### macOS 安裝

1. 下載 `ContractRiskManagement-darwin-x64-1.0.0.zip`
2. 解壓縮文件
3. 將 `ContractRiskManagement.app` 拖到「應用程序」文件夾
4. 首次運行時，右鍵點擊應用選擇「打開」

### Linux 安裝

**Debian/Ubuntu**:
```bash
sudo dpkg -i contract-risk-management_1.0.0_amd64.deb
```

**RedHat/Fedora**:
```bash
sudo rpm -i contract-risk-management-1.0.0-1.x86_64.rpm
```

## 快速開始

### 首次登錄

1. 啟動應用程序
2. 使用默認管理員賬號登錄:
   - 用戶名: `admin`
   - 密碼: `admin123`

> ⚠️ **安全提示**: 首次登錄後，請立即修改默認密碼！

### 創建第一個項目

1. 點擊「創建項目」按鈕
2. 輸入項目名稱（例如：「2024年度採購合同」）
3. 輸入項目描述（可選）
4. 點擊「確定」

### 導入合同文檔

1. 進入項目詳情頁面
2. 點擊「導入文檔」按鈕
3. 選擇合同文件（支持 DOCX、PDF、TXT）
4. 等待文檔解析完成

### 查看風險分析

1. 文檔導入後，系統自動進行風險分析
2. 點擊「查看詳情」進入文檔審閱頁面
3. 風險條款會以不同顏色標註：
   - 🔴 **紅色**: 高風險
   - 🟡 **黃色**: 中風險
   - 🔵 **藍色**: 低風險

## 功能詳解

### 1. 用戶管理

#### 登錄
- 輸入用戶名和密碼
- 勾選「記住我」可保持登錄狀態
- 登錄失敗 5 次後賬號將被鎖定 10 分鐘

#### 修改密碼
1. 點擊右上角用戶頭像
2. 選擇「設置」
3. 在「安全設置」中修改密碼
4. 新密碼要求：至少 8 位，包含字母和數字

#### 登出
- 點擊右上角用戶頭像
- 選擇「登出」

### 2. 項目管理

#### 創建項目
1. 在項目列表頁面點擊「創建項目」
2. 填寫項目信息：
   - **項目名稱**（必填，1-100 字符）
   - **項目描述**（可選，最多 500 字符）
3. 點擊「確定」創建

#### 編輯項目
1. 在項目列表中找到要編輯的項目
2. 點擊「編輯」按鈕
3. 修改項目信息
4. 點擊「保存」

#### 刪除項目
1. 點擊項目的「刪除」按鈕
2. 確認刪除操作
3. ⚠️ **注意**: 刪除項目會同時刪除該項目下的所有文檔

#### 查看項目統計
- 項目列表顯示每個項目的文檔數量
- 點擊項目名稱查看詳細統計

### 3. 文檔管理

#### 導入文檔

**支持的格式**:
- Microsoft Word (.docx)
- PDF (.pdf)
- 純文本 (.txt)

**導入步驟**:
1. 進入項目詳情頁面
2. 點擊「導入文檔」
3. 選擇文件（支持拖拽上傳）
4. 等待解析完成

**文檔狀態**:
- 🔄 **處理中**: 正在解析文檔
- ✅ **已完成**: 解析成功，可以查看
- ❌ **失敗**: 解析失敗，請檢查文件格式

#### 查看文檔
1. 在文檔列表中點擊「查看」
2. 進入文檔審閱頁面
3. 左側顯示條款列表
4. 右側顯示風險統計和詳情

#### 刪除文檔
1. 點擊文檔的「刪除」按鈕
2. 確認刪除操作

### 4. 風險識別

#### 自動風險分析

系統會自動識別以下類型的風險：

**高風險條款**:
- 預付款條款
- 無限責任條款
- 知識產權轉讓條款
- 單方變更條款

**中風險條款**:
- 管轄地條款
- 保密期限條款
- 自動續約條款

**低風險條款**:
- 不可抗力條款

#### 手動調整風險等級
1. 點擊風險條款
2. 在詳情面板中選擇新的風險等級
3. 系統自動保存修改

#### 查看風險統計
- 右側面板顯示風險統計：
  - 總風險數量
  - 高/中/低風險分布
  - 風險百分比

### 5. 批註功能

#### 添加批註
1. 選中需要批註的條款
2. 點擊「添加批註」按鈕
3. 輸入批註內容
4. 點擊「保存」

#### 查看批註
- 有批註的條款會顯示批註圖標
- 點擊圖標查看批註內容

#### 編輯/刪除批註
1. 點擊批註圖標
2. 選擇「編輯」或「刪除」
3. 確認操作

### 6. 報告導出

#### 導出 PDF 報告
1. 在文檔審閱頁面點擊「導出報告」
2. 選擇保存位置
3. 等待生成完成

**報告內容包括**:
- 文檔基本信息
- 風險統計圖表
- 詳細風險列表
- 風險條款原文
- 風險描述和建議

#### 導出 Word 文檔
1. 點擊「導出 DOCX」
2. 選擇保存位置
3. 導出的文檔包含所有批註

### 7. 版本管理

#### 查看版本歷史
1. 在文檔詳情頁面點擊「版本歷史」
2. 查看所有歷史版本
3. 每個版本顯示：
   - 修改時間
   - 修改人
   - 修改說明

#### 對比版本
1. 選擇兩個版本
2. 點擊「對比」
3. 查看差異高亮顯示

#### 回滾版本
1. 選擇要回滾的版本
2. 點擊「回滾」
3. 確認操作

### 8. 設置

#### 系統設置
- **語言設置**: 選擇界面語言（中文/英文）
- **主題設置**: 選擇淺色/深色主題
- **通知設置**: 開啟/關閉系統通知

#### 風險規則管理
1. 進入「設置 > 風險規則」
2. 查看所有風險識別規則
3. 可以添加/編輯/刪除規則

#### 數據管理
- **清理緩存**: 清理臨時文件
- **數據備份**: 備份數據庫
- **數據恢復**: 從備份恢復數據

## 常見問題

### Q1: 忘記密碼怎麼辦？

**A**: 請聯繫系統管理員重置密碼。管理員可以在用戶管理頁面重置任何用戶的密碼。

### Q2: 為什麼文檔導入失敗？

**A**: 可能的原因：
1. 文件格式不支持（只支持 DOCX、PDF、TXT）
2. 文件損壞或加密
3. 文件過大（建議 < 10MB）
4. 文件路徑包含特殊字符

**解決方法**:
- 檢查文件格式
- 嘗試重新保存文件
- 減小文件大小
- 重命名文件（使用英文名稱）

### Q3: 風險識別不準確怎麼辦？

**A**: 
1. 可以手動調整風險等級
2. 在設置中添加自定義風險規則
3. 提供反饋給開發團隊改進算法

### Q4: 如何提高風險識別準確率？

**A**:
1. 使用清晰的文檔格式
2. 避免使用掃描版 PDF
3. 確保文檔文字可選中
4. 定期更新風險規則庫

### Q5: 數據存儲在哪裡？

**A**: 
- Windows: `C:\Users\<用戶名>\AppData\Roaming\contract-risk-management\`
- macOS: `~/Library/Application Support/contract-risk-management/`
- Linux: `~/.config/contract-risk-management/`

### Q6: 如何備份數據？

**A**:
1. 進入「設置 > 數據管理」
2. 點擊「備份數據」
3. 選擇備份位置
4. 定期備份以防數據丟失

### Q7: 可以同時打開多個項目嗎？

**A**: 可以。系統支持多項目並行管理，但建議一次只審閱一個文檔以保證效率。

### Q8: 導出的報告可以編輯嗎？

**A**: 
- PDF 報告: 不可直接編輯，但可以使用 PDF 編輯器
- Word 文檔: 可以直接編輯

### Q9: 系統支持多用戶協作嗎？

**A**: 當前版本為單機版，不支持多用戶實時協作。企業版將支持團隊協作功能。

### Q10: 如何更新系統？

**A**: 
1. 系統會自動檢查更新
2. 有新版本時會提示下載
3. 也可以手動下載最新版本安裝

### 文件格式說明

**DOCX**:
- 最佳支持格式
- 保留完整格式信息
- 支持批註和修訂

**PDF**:
- 支持文字型 PDF
- 不支持掃描版 PDF
- 建議使用 OCR 處理掃描版

**TXT**:
- 純文本格式
- 無格式信息
- 適合簡單合同

### 術語表

- **項目**: 用於組織管理多個相關合同文檔的容器
- **文檔**: 導入系統的合同文件
- **條款**: 文檔中的單個條款或段落
- **風險**: 條款中識別出的潛在法律或商業風險
- **批註**: 用戶對條款添加的註釋和建議
- **版本**: 文檔的歷史修改記錄

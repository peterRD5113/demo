## 🎯 項目概述

### 核心功能

合同風險管理系統是一款智能化的桌面應用程序，主要功能包括：

1. **智能風險識別**: 自動掃描合同條款，識別高、中、低風險
2. **多格式支持**: 支持 DOCX、PDF、TXT 等多種文檔格式
3. **項目管理**: 按項目組織管理多個合同文檔
4. **批註功能**: 為風險條款添加批註和建議
5. **報告導出**: 生成專業的風險分析報告（PDF 格式）
6. **版本管理**: 追蹤文檔修改歷史
7. **用戶權限**: 支持管理員和普通用戶角色

### 技術架構

- **前端**: React 18 + TypeScript + Ant Design 5
- **後端**: Electron 28 + Node.js 18 + TypeScript
- **數據庫**: SQLite (Better-SQLite3)
- **打包**: Electron Forge 7
- **測試**: Jest 29

## 🎓 技術難點與解決方案

### 1. Electron 進程通信

**難點**: 主進程和渲染進程之間的安全通信

**解決方案**:
- 使用 Preload 腳本暴露安全的 API
- 定義清晰的 IPC 通道
- 類型安全的請求/響應

### 2. 數據庫設計

**難點**: 複雜的數據關係和查詢優化

**解決方案**:
- 合理的表結構設計
- 15+ 個索引優化查詢
- Repository 模式封裝數據訪問

### 3. 風險識別算法

**難點**: 準確識別合同中的風險條款

**解決方案**:
- 關鍵詞匹配算法
- 可配置的風險規則
- 支持自定義規則

### 4. 文檔解析

**難點**: 支持多種文檔格式

**解決方案**:
- 使用 mammoth 解析 DOCX
- 使用 pdf-lib 解析 PDF
- 統一的文檔解析接口

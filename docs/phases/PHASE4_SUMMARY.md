### 關鍵文件位置
- 應用入口: `src/renderer/index.tsx`
- 根組件: `src/renderer/App.tsx`
- 認證上下文: `src/renderer/contexts/AuthContext.tsx`

### 頁面路由
- `/login` - 登錄頁面
- `/projects` - 項目列表
- `/project/:projectId/documents` - 文檔列表
- `/project/:projectId/document/:documentId` - 文檔審閱
- `/settings` - 設置頁面

### 使用示例

#### 使用 AuthContext
```typescript
import { useAuth } from '../contexts/AuthContext';

const MyComponent = () => {
  const { user, token, login, logout } = useAuth();
  
  // 登錄
  await login('username', 'password');
  
  // 登出
  logout();
};
```

#### 調用 IPC
```typescript
// 獲取項目列表
const response = await window.electronAPI.project.list({
  token,
  page: 1,
  pageSize: 10,
});

if (response.success) {
  console.log(response.data.projects);
}
```

### 注意事項
1. 所有 IPC 調用都需要傳遞 token
2. 使用 message 組件統一提示用戶
3. 異步操作要顯示 loading 狀態
4. 刪除操作需要二次確認
5. 表單提交前要驗證輸入

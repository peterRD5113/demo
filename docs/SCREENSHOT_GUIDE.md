## 🔧 如何啟動應用程序

### 方法 1: 開發模式啟動

```bash
# 1. 進入桌面應用目錄
cd desktop

# 2. 確保依賴已安裝
npm install

# 3. 啟動應用程序
npm start
```

### 方法 2: 構建後啟動

```bash
# 1. 構建應用程序
cd desktop
npm run build

# 2. 啟動構建後的應用
npm run start:prod
```

## ⚠️ 注意事項

### 1. 數據庫準備
在啟動應用前，確保數據庫已初始化：

```bash
# 檢查數據庫文件
ls desktop/database.sqlite

# 如果不存在，應用會自動創建
```

### 2. 測試數據
應用會自動創建測試賬號：

| 角色 | 用戶名 | 密碼 |
|------|--------|------|
| 管理員 | admin | admin123 |
| 普通用戶 | user | user123 |

### 3. 截圖工具
推薦使用：
- Windows: Snipping Tool / Win + Shift + S
- 截圖後保存為 PNG 格式
- 確保截圖清晰，文字可讀

### 4. 隱私保護
截圖時注意：
- ✅ 可以顯示測試數據
- ❌ 不要包含真實的個人信息
- ❌ 不要包含敏感的系統路徑

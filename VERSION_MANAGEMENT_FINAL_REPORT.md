## 🎯 实现的完整功能流程

### 用户操作流程
```
1. 用户在文档审阅页面修改条款
   ↓
2. 点击"保存版本"按钮
   ↓
3. 系统创建版本快照（所有条款）
   ↓
4. 显示"版本保存成功"提示
   ↓
5. 继续修改条款
   ↓
6. 点击"对照修订"按钮
   ↓
7. 页面切换到双栏视图
   ↓
8. 左侧显示最新保存版本（只读）
   右侧显示当前编辑内容（可编辑）
   ↓
9. 可以同步滚动查看差异
   ↓
10. 可以继续编辑右侧内容
    ↓
11. 点击"保存版本"创建新快照
    ↓
12. 点击"退出对照"返回单栏模式
```

## 🔧 技术实现

### API 调用
```typescript
// 创建版本
window.electronAPI.version.create(documentId, userId, description)

// 获取最新版本
window.electronAPI.version.getLatest(documentId, userId)

// 获取版本条款
window.electronAPI.version.getClauses(versionId, userId)
```

### 关键技术点
1. **React Hooks**
   - `useState`: 状态管理
   - `useRef`: 滚动容器引用
   - `useCallback`: 滚动函数优化

2. **滚动同步**
   - `requestAnimationFrame`: 性能优化
   - `isSyncing` 标志位: 防止循环触发
   - `setTimeout`: 延迟重置标志

3. **条件渲染**
   - 单栏/双栏模式切换
   - 目录显示/隐藏
   - 按钮状态变化
